# Sample Requirements

REQ-CTRL-001: При запуске система должна выполнить self-test памяти.
REQ-CTRL-002: Если self-test памяти завершился ошибкой, система должна установить error_flag.
REQ-CTRL-003: При активном error_flag система должна игнорировать команды управления.
REQ-CTRL-004: При успешном self-test система должна перейти в normal mode.
REQ-CTRL-005: Система должна ограничивать команду управления допустимым диапазоном значений.
REQ-CTRL-006: Система должна сохранять последнюю принятую команду управления.
